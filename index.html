Getting Started

Welcome to your new HostBurro site. You can manage your project by visiting the admin console at https://hostburro.com/admin or by going directly to the project page at https://hostburro.com/admin/your-project-name. The contents of this site were synchronized from https://github.com/HostBurro/getting-started.

If you have files for your website all ready to go in a public git repository, just head over to your project settings now and input the git URL there. Once you tap the "Sync project" button, HostBurro will synchronize your git repo to HostBurro Serving and your site will be ready to go. Synchronization should happen very quickly but it may take around ten minutes before HTTP caches update with your new files.

If you do not have a website ready to go, we'd recommend you fork this project. You can follow this guide to do so. Once you've done that, you can sync from a repository all your own and start changing your site to suit your needs.

Just like most web servers, the default URL of your-project-name.hburro.com/ serves an index.html file if it is available. If a requested page does not exist in your provided site files, the 404.html file from the repository will be served (if provided). If a 404.html file isn't provided, a default one will be served for you.
